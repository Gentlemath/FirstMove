# First Move Starter

Build Your Daily Momentum

This is a starter project structure for the First Move prototype.

## Suggested build order
1. Put your workflow screenshots into `docs/assets/`
2. Keep your spec in `docs/`
3. Build the static homepage first in `frontend/`
4. Add backend endpoints in `backend/`
5. Connect calendar, then LLM generation

## Main folders
- `docs/` product spec, graphs, notes
- `frontend/` UI prototype
- `backend/` API and LLM logic
